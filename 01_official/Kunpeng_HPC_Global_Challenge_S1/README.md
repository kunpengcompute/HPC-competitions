# 鲲鹏高性能计算全球挑战赛（S1赛季）
## 🏆 赛事简介
鲲鹏高性能计算全球挑战赛是面向高性能计算行业基于鲲鹏计算平台的并行计算工程挑战赛事，旨在培养参赛人员的并行计算思想和并行计算系统能力，秉承以赛促学，以赛促教，以赛促用的办赛理念，推动建立鲲鹏超算新生态，促进开发者能力提升。

## 📅 赛事时间
- 报名时间：2025年12月15日 —— 2026年4月30日
- 作品提交：2025年12月15日 —— 2026年4月30日
- 颁奖日期：2026年5月23日

## 🎯 赛题方向
1. FFT优化
2. Tensor contraction优化
